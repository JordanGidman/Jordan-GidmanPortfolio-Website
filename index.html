Basket Application

It allows the user to add any of the video game keys you see on offer to a cart which tracks the number of items that have been added and displays it to screen it will also calculate a total cost for the basket and users can add and remove items and the total will remain up to date. This was my first "real world" application and i learned a lot about the DOM and event bubbling. i have also added sorting buttons which can be seen in the second image that are functional and will sort based on price. The far right image is the responsive basket layout.

Quit Smoking Helper App

This is a responsive single page JavaScript application that i built to aid people in quitting smoking by taking a user given number of days since their last smoke and based on that will generate suggestions deemed sufficient for the users time without to help fight cravings and stick with it. The first screen shot shows the home and user input page.

The second shows the suggestions as well as each button having a change suggestion button and giving the user the option to increase or decrease the level of strength the suggestions are. e.g recent quitters may need stronger more hands on suggestions but if they just want a quick fix they can decrease the strength.
