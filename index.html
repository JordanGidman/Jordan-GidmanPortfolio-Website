React Speed Typing App

This is a React speed typing test application. It is simple in nature but i feel uses a good amount of react skills and concepts. This application uses a react library to generate random words which the user is tasked with typing as many of as they can. for me the most difficult part of this application was the text effect. Being that i needed a way for the user to give input and to do this i have simply positioned the words, the input and the text display which is the text that shows after the user has typed either red or white depending on if the letter is correct or not. For the timer i have utilised a useEffect that calls a setInterval and setTimeout to manage the countdown and allow for the words per minute calculations. This is V1 as i plan to remake this with some chart libraries for the results and i feel the typing needs to be handled in another way but i am aware i would need the use of custom hooks which i have not yet gotten in to.

Freelance Client Gallery Site

This is a fully responsive gallery website that i built freelance for a digital art and photography creator. i utilised set interval to give the header a sliding image that will alternate every 3 seconds. the client wanted the primary gallery's images to be the same size to combat the image stretching when using smaller viewports i have added them all as a background of a div. Allowing for me to achieve the same size for images of a different orientation but at the cost of some of the image being slightly trimmed(the client agreed to this). Initally i had used firebase to store and pull the images however the client did not wish to have her images stored in this way and so i have enabled the client to simply drag and drop images into a folder and rename them to update the main and preview galleries or the hero sliding images. So to change an image the client simply drops a new image in the folder and renames it to the one they want replaced and job done it will be replaced or added. i have done this by having class methods to render the images dynamically rather than hardcoding them. the user can also view any image in as close to full resolution as their viewport allows by clicking the image itself. This is not the final version and is currently a work in progress but it is close enough to completion that i feel comfortable putting it here. Countdown API Word Game

This is a responsive single page JavaScript application that is isnpired by the tvshow countdown. The user is given a string of random letters and they are tasked with making the longest word they can from those letters in 30 seconds. The app is plugged in to a dictionary API as part of the checks used to ensure the word given by the user is valid. Valid being that the word is a real word and also consists only of letters shown that round. I have also ensured that the highscore is persistent for the user. As i am storing only one variable i have not used a database but instead i am utilizing the browsers localStorage
